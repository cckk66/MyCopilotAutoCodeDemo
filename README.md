# MyCopilotAutoCodeDemo

This repository contains a .NET 8 console application that outputs "Hello, World!".

## Prerequisites

- .NET 8.0 SDK or later

## Running the Application

Navigate to the HelloWorld directory and run:

```bash
cd HelloWorld
dotnet run
```

Or build and run separately:

```bash
cd HelloWorld
dotnet build
dotnet run
```

## Project Structure

- `HelloWorld/` - .NET 8 console application
  - `Program.cs` - Main program file
  - `HelloWorld.csproj` - Project configuration file
